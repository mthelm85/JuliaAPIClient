# Julia API Client

This is a simple proof-of-concept to show how to design a client app with Vue.js that calls an API written in Julia.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
